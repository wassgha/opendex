What it does

01

Voice-first loop

Wake word → listen → think (with tools) → speak. Natural follow-ups, plus opt-in barge-in to interrupt mid-reply.

02

Any model

Routes through the Vercel AI Gateway, so one key gets you Claude, GPT, Gemini, and more.

03

Free & offline option

Vosk wake word + local Whisper transcription (WASM, no signup) and your OS's built-in voice. Nothing leaves the machine except the LLM call.

04

Pluggable voice I/O

Wake via push-to-talk, Vosk, Porcupine, or Web Speech; transcribe locally or in the cloud; speak via ElevenLabs or system TTS.

05

Full-interface themes

The theme is the whole UI: a cinematic Jarvis HUD, a minimal Talking Dot, a Typing Cursor terminal, or a warm Editorial layout — all reactive to your voice.

06

Agentic skills, gated

The agent takes real actions (open apps & URLs). Sensitive ones hit an Allow once / Always / Deny prompt, remembered per skill.

07

Computer-use (opt-in)

Let it see the screen and drive the mouse & keyboard to operate apps for you. Works with any vision model, behind the permission gate.

08

Secure by design

API keys are encrypted with your OS keychain and live only in the main process — never in the UI.

Pick your interface

A theme renders the entire experience — visualization, transcript, and controls. Switch any time.
